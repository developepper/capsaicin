# T15: `capsaicin ticket run`

**Phase**: 5 — Implementation Pipeline
**Type**: Implementation
**Risk**: First end-to-end pipeline — will surface integration issues

## Goal

Full implementation pipeline from ticket selection through post-run state
transition.

## Scope

- `capsaicin ticket run [TICKET_ID]`
- Auto-select: next `ready` ticket ordered by `created_at` with all deps `done`
- Cycle-limit shortcut: if ticket in `revise` and `current_cycle >= max_cycles`,
  transition directly to `human-gate` with `gate_reason='cycle_limit'` (no
  adapter invocation)
- Transition to `implementing`
- Update orchestrator state (active ticket/run, status=running)
- If from `ready`: `init_cycle`; if from `revise`: `increment_cycle`
- Assemble implementer RunRequest via prompt builder
- Insert `agent_runs` row with `exit_status='running'` before invocation
- Invoke implementer adapter
- Update `agent_runs` with terminal status, duration, stdout/stderr,
  adapter_metadata
- Capture post-run diff
- Success + non-empty diff -> persist `run_diffs`, transition to `in-review`
- Success + empty diff -> transition to `human-gate`
  (`gate_reason='empty_implementation'`), set orchestrator `awaiting_human`
- Failure/timeout -> increment impl retry; if under limit retry (re-invoke); if
  over limit transition to `blocked` (`blocked_reason='implementation_failure'`)
- Log events to `activity.log`: `RUN_START`, `RUN_END`, state transitions

## Non-goals

- No review invocation
- No loop automation

## Acceptance criteria

- Auto-selection picks correct ticket respecting dependencies and ordering
- Cycle-limit shortcut works without invoking adapter
- Success + changes -> `in-review`
- Success + no changes -> `human-gate` with `empty_implementation`
- Failure -> retry counter incremented
- Retry limit exceeded -> `blocked`
- `agent_runs` row inserted before and updated after invocation
- `run_diffs` row created on success with non-empty diff
- `orchestrator_state` reflects active ticket/run during execution
- `state_transitions` rows recorded

# T02: Database module

**Phase**: 1 — Foundation
**Type**: Implementation

## Goal

Reusable SQLite connection factory and migration runner.

## Scope

- `src/capsaicin/db.py`
- `get_connection(db_path) -> sqlite3.Connection` — enables
  `PRAGMA foreign_keys = ON` on every connection
- `run_migrations(conn)` — executes schema SQL; idempotent via
  `CREATE TABLE IF NOT EXISTS` or a version table
- Tests: FK enforcement raises on violation; `run_migrations` is idempotent
  (call twice, no error)

## Non-goals

- No schema SQL yet (that's T03)
- No config integration

## Acceptance criteria

- `get_connection` returns a connection where FK violations raise errors
- `run_migrations` can be called twice without error
- Tests pass against in-memory SQLite

# T01: Python project scaffolding

**Phase**: 1 — Foundation
**Type**: Implementation

## Goal

Runnable `capsaicin` CLI entrypoint with no commands yet.

## Scope

- `pyproject.toml`: project metadata, Python >=3.11, dependencies (`click`,
  `python-ulid`, `pytest` dev dependency, `ruff` dev dependency)
- `src/capsaicin/` package with `__init__.py`
- `src/capsaicin/cli.py` with a `click.Group` entrypoint
- `capsaicin` console_scripts entrypoint in pyproject.toml
- One test: `capsaicin --help` exits 0 and prints usage
- Initial GitHub Actions workflow at `.github/workflows/ci.yml`:
  - docs/meta check for required top-level docs and ticket docs directory
  - Python 3.11 setup
  - editable install
  - `ruff format --check`
  - `capsaicin --help`
  - `pytest`

## Non-goals

- No commands
- No database
- No config

## Acceptance criteria

- `pip install -e .` succeeds
- `capsaicin --help` runs and prints usage
- `pytest` discovers and passes >= 1 test
- CI workflow runs on push and pull_request and passes with:
  - docs/meta checks
  - `ruff format --check`
  - `capsaicin --help`
  - `pytest`

## Dependencies

None.

## Notes

Use `click`, not `typer`. Use `tomllib` (stdlib 3.11+) for TOML. Add
`python-ulid` to dependencies now so it's available for T06. Use Ruff as the
formatter from the start so formatting checks are stable before more code lands.

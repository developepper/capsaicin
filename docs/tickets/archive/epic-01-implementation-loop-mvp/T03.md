# T03: Schema migration

**Phase**: 1 — Foundation
**Type**: Implementation
**Status**: Complete

## Goal

Full MVP schema and indexes as a runnable migration.

## Scope

- Migration SQL with all 11 tables from `data-model.md`: `projects`, `tickets`,
  `acceptance_criteria`, `ticket_dependencies`, `agent_runs`, `run_diffs`,
  `review_baselines`, `findings`, `orchestrator_state`, `state_transitions`,
  `decisions`
- All 10 indexes from `data-model.md`
- Tables created in FK dependency order
- Tests: all tables exist post-migration; CHECK constraints enforced (invalid
  status rejected); FK constraints enforced (invalid FK insert fails)

## Non-goals

- No data access layer
- No ORM

## Acceptance criteria

- Migration creates all 11 tables and 10 indexes
- CHECK constraints reject invalid values (e.g., `tickets.status = 'invalid'`)
- FK constraints reject invalid references
- Migration is idempotent

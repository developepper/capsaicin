# T23: `capsaicin ticket defer`

**Phase**: 7 — Human Gate & Decisions
**Type**: Implementation
**Suggested PR pairing**: Ship with T21, T22, T24 (all human-gate commands)

## Goal

Defer or abandon a ticket from human-gate.

## Scope

- `capsaicin ticket defer [TICKET_ID] [--rationale TEXT] [--abandon]`
- Accept only from `human-gate`
- Without `--abandon`: `defer` decision, transition to `blocked` with
  human-readable `blocked_reason`
- With `--abandon`: `reject` decision, transition to `done`
- Set orchestrator status to `idle`

## Non-goals

- No automatic blocked-ticket recovery

## Acceptance criteria

- Defer without abandon -> `blocked`
- Defer with abandon -> `done`
- Correct decision type recorded (`defer` vs `reject`)
- `blocked_reason` set on defer
- Tickets not in `human-gate` rejected
- Orchestrator -> `idle`

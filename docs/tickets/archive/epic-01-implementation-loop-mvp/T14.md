# T14: Diff capture module

**Phase**: 5 — Implementation Pipeline
**Type**: Implementation

## Goal

Capture tracked-file diffs and persist them.

## Scope

- `src/capsaicin/diff.py`
- `capture_diff(repo_path) -> DiffResult` — run `git diff HEAD`, return diff
  text + list of changed files
- `persist_run_diff(conn, run_id, diff_result)` — insert into `run_diffs`
- `get_run_diff(conn, run_id) -> DiffResult` — retrieve persisted diff
- `diffs_match(a: str, b: str) -> bool` — compare two diff texts
- DiffResult: dataclass with `diff_text: str`, `files_changed: list[str]`,
  `is_empty: bool`
- Tests: use a temporary git repo with controlled file modifications

## Non-goals

- No untracked file handling
- No review baseline (T16)

## Acceptance criteria

- `capture_diff` returns correct diff text for modified tracked files
- Empty diff detected (no changes -> `is_empty=True`)
- Diff persisted and retrievable by run_id
- `diffs_match` correctly identifies matching and divergent diffs
